# MyTube Clone

A simple YouTube-inspired website built using HTML, CSS, and JavaScript.

## Features

* YouTube-style homepage
* Navigation bar with search box
* Video thumbnails with images
* Clean and responsive design

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Clone or download this repository.
2. Open the project folder.
3. Run index.html in your browser.
